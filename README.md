# Recipe Website (ASP.NET & SQL Server) 🍽️

## 📌 Project Description
This project is a web-based recipe website developed using ASP.NET Web Forms and SQL Server.

## 🧠 Technologies Used
- ASP.NET Web Forms (.aspx)  
- C#  
- SQL Server  
- HTML / CSS  

## 📊 Features
- Recipe listing by categories  
- Add new recipes  
- Admin panel for managing content  
- User recipe suggestion system  
- Contact/message system  
- Dynamic content from database  

## 🎯 Goal
To develop a dynamic web application for sharing and managing food recipes.

## 🚀 Status
Completed as a full-stack web development project.
